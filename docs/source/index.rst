Welcome to p5controls's documentation!
======================================

This package helps you with controlling your measurement setup using python. It provides you with useful structurs which lets you focus on implementing device specific drivers and performing your experiments, while allowing for easy data acquisition and control. In addition, qt-based gui elements are provided to allow for easy creation of a gui with live plotting among its features.
